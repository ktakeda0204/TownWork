Homebrew（ほーむぶるー）

Mac OSでソフトウェアのパッケージ群のインストールを管理するためのシステム。パッケージ管理システム。 Linuxのyumとかapt-getの類。自分はhomebrewを使っている。（MaCOSは他にMacPortsというパッケージマネージャーもある。）

pyenv

様々なバージョンのPythonを管理するツール
  • pyenvをインストールする
    brew install pyenv
  • pyenvのバージョンを確認
    pyenv versions
  • バージョンの切り替え
    pyenv global <パッケージ名>

Anaconda

Python本体に加え，科学計算やデータ分析などに使えるライブラリがセットで入っているパッケージ. Anacondaを直にインストールするとhomebrewと競合するため、pyenvでインストールした。

テキスとファイルの読み込み

  • read()：ファイルの中身を全て読み込んで文字列を返す
  • readline()：ファイルを1行ずつ読み込んで文字列を返す
  • readlines()：ファイルを1行ずつ読み込んで、リストを返す
一行ずつ読んで、何かの条件をかけて処理していく。 たんにopen()をfor文に喰わせれば良い。これで一行ずつ読みこんだ結果がprintされる。 

    for line in open("line.txt", "r"):
      print(line)
