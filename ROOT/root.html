How to use ROOT

  • Scan()オプション
  • Draw()
  • マクロに引数を渡す

Scan()オプション

Scan("変数名：変数名：...","条件式","オプション")
よく使うのはオプションに"colsize=20"とかやって表示幅を増やしたりする。

Draw()

ROOTのインタープリター上で簡易的にヒストグラムをDraw出来る。 パッと生分布がどうなっているかを確認したい時に便利。 (TTreeの名前)-Draw("変数名") とする事で、パッと分布を見れる。 (TTeee)-Draw("変数名","条件式","colz") とかも出来る

マクロに引数を渡す

マクロに引数を渡す事が出来る。 

    root -l 'マクロ名("引数")'
